Leukemia Classification

นี่คือโปรเจกต์เล็ก ๆ สำหรับวิชา Machine Learning
เว็บแอปนี้ใช้เทคโนโลยี deep learning ในการตรวจจับเซลล์มะเร็งเม็ดเลือดขาวเฉียบพลันชนิดลิมโฟบลาสติก (ALL)
ซึ่งเป็นมะเร็งในวัยเด็กที่พบบ่อยที่สุด คิดเป็นประมาณ 25% ของมะเร็งในเด็ก

ภาพรวมโครงการ

โครงการนี้มุ่งหวังที่จะทำนายว่าภาพเซลล์ที่นำเข้ามามีสัญญาณของมะเร็ง (ALL) หรือไม่ (HEM) โดยใช้โมเดล deep learning ที่ได้รับการฝึกจากชุดข้อมูลภาพทางการแพทย์

เครื่องมือที่ใช้
  • - Python, TensorFlow
  • - Flask สำหรับ API และเว็บเซิร์ฟ
  • - Bootstrap 5 สำหรับออกแบบ responsive
  • - Pillow สำหรับประมวลผลภาพ
วิธีการใช้งาน
  1. คลิกที่เมนู "Upload" หรือปุ่ม "อัปโหลด"
  2. เลือกไฟล์ภาพเซลล์ (รองรับ JPEG/PNG)
  3. กดปุ่ม Submit แล้วรอการแสดงผลการทำนาย
